## 🚀 Getting Started

ZGameLib is a personal game library for Windows. It keeps your Steam, Epic, GOG, and custom games in one place. It also tracks playtime, ratings, and time to beat.

Use it to start games fast, sort your library, and keep your game list local on your PC. It works without an account.

## 📥 Download and Install

1. Open the download page: [ZGameLib on GitHub](https://github.com/acornshaped-oysterdressing296/ZGameLib)
2. Find the latest Windows release or build on the page.
3. Download the Windows app file.
4. Open the file to start the app.
5. If Windows asks for permission, choose Run.

If you use a portable copy, you can place the app in any folder and run it from there. It does not need a cloud profile or a sign-in.

## 🖥️ What You Need

ZGameLib is made for Windows PCs.

- Windows 10 or Windows 11
- A modern CPU
- 4 GB RAM or more
- A few hundred MB of free disk space
- Steam, Epic, or GOG accounts if you want auto-scan support

It also works well with custom game folders and local game shortcuts.

## 🎯 What ZGameLib Does

ZGameLib helps you keep track of your games without switching between launchers.

- Auto-scans Steam, Epic, and GOG games
- Adds custom games from folders or shortcuts
- Tracks playtime for each game
- Stores ratings so you can sort your best games
- Shows HowLongToBeat time-to-beat data
- Supports a mod loader for games that need extra files
- Includes a spin wheel for picking what to play
- Offers 7 themes, including a dark theme
- Exports your library to CSV
- Supports portable mode for local use

The app keeps your data on your machine. It does not use an account or online profile.

## 🎮 Add Games to Your Library

ZGameLib supports three main ways to build your library.

### Steam
It can find installed Steam games and add them to your list.

### Epic
It can scan your Epic library and show installed titles.

### GOG
It can pick up games from GOG installs and add them too.

### Custom Games
You can add any Windows game, even if it is not from a store. This works for older games, indie games, and games stored in a folder on your PC.

## 🔧 How It Works

ZGameLib uses a local database to store your library data. It reads game details from your installed launchers and from the entries you add by hand.

The app uses:

- Tauri 2 for the desktop app shell
- React for the user interface
- Rust for the core app logic
- SQLite for local data storage
- Windows support as the main target

You do not need to manage these tools yourself. They are part of how the app is built.

## 🧰 Basic Troubleshooting

If the app does not start:

- Make sure you downloaded the Windows build
- Try opening the file again with admin rights
- Check that Windows did not block the file
- Confirm your antivirus did not move it
- Reopen the app after installing your game launchers

If a game does not appear:

- Check that the launcher is installed
- Open the app again and rescan
- Add the game as a custom entry if needed

If playtime or beat-time data looks wrong:

- Refresh the library
- Make sure the game entry points to the right install
- Reopen the app after your launcher updates its data
